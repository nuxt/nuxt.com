## A new Search

You may notice a new modal when hitting :kbd[CMD+K], we leverave Nuxt UI [`<CommandPalette>`](https://ui.nuxt.com/navigation/command-palette) mixed with Nuxt Content data (search & navigation) to provide a better search experience.

With the command palette, you can quickly:
- Jump to a page
- Search in the documentation
- Search a module
- Switch the color mode

We plan to add more commands in the future.

## Migration to Nuxt UI

The new website is powered by [Nuxt UI](https://ui.nuxt.com), our UI library tailored made for Nuxt and built on top of [Tailwind CSS](https://tailwindcss.com) & [Headless UI](https://headlessui.com/).

The website is also using **Nuxt UI Pro**, a set of premium components built on top of Nuxt UI to create beautiful & responsive Nuxt applications in minutes.

It includes components such as `<UHeader>`, `<UFooter>`, `<ULandingHero>`, `<ULandingCard>` and more.

::callout{icon="i-ph-info-duotone" color="blue"}
We plan to launch the full documentation of Nuxt UI Pro end of October. If you cannot wait and want early access, you can already [purchase a license](https://ui.nuxt.com/pro/purchase) now and get access to our private repository on GitHub.
::

This [migration](https://github.com/nuxt/nuxt.com/pull/1365) was a great opportunity to improve Nuxt UI & UI Pro and fix some bugs, as well a difference of [+8,448]{.text-green-500} / [-23,040]{.text-red-500} lines of code changed.
